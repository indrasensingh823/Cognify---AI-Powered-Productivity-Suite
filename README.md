# 🚀 Cognify – AI Powered Productivity Suite

> Supercharge your team’s productivity with AI 🚀

Cognify is a **modern AI SaaS platform** that combines powerful **AI tools + project management** into one seamless experience. Built for **developers, agencies, and modern teams**, it helps automate workflows, generate content, and boost productivity.

---

## 🌐 Live Demo

🔗 [Click Here](https://cognify-ai-powered-productivity-sui.vercel.app/)

## ✨ Features

### 🧠 AI Tools

* ✍️ AI Article & Blog Generator  
* 🎨 AI Image Generator  
* 🪄 Background & Object Remover  
* 📄 Resume Analyzer  

### 📊 Productivity Tools

* 📁 Multi-Tenant Workspaces  
* ✅ Project & Task Management  
* 📅 Interactive Calendar  
* 📈 Advanced Analytics Dashboard  

### 🌍 Website & Funnel Builder

* Drag & Drop Builder  
* Client Funnel Management  
* Custom Branding Support

## Installation & Setup
### 1️⃣ Clone the Repository
```bash
git clone https://github.com/indrasensingh823/cognify.git
cd cognify
```
### 2️⃣ Install Dependencies
```bash
npm install
```
### 3️⃣ Run Locally
```bash
npm run dev
```
```bash
👉 Open: http://localhost:3000
```
